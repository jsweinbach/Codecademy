Common HTML attributes

File Type Tag Description
HTML <a> Anchor tag, used to specify the text that is the "anchor" for a link.
HTML <p> Paragraph tag, contains a block of plain text
HTML <href> Hyperlink REFerence, specifies the URL or address on the web.
HTML <id> Identifies an HTML element
HTML <style> Defines the style of an HTML element and can alter default styles
HTML <class> Assigns an element to a class or multiple classes
